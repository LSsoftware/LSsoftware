照例的第一篇博客---如何使用Hexo搭建博客

注：操作简单就不放图了

为何搭建Hexo博客

博主本来是个不写博客的人，懒人一个，大写的拿来主义。

写博客的认为原因就不追究了，下面说说为什么要选择Hexo博客吧。

原因：

1、自己购买控件要钱(域名购买:阿里云，百度云，狗爹。空间购买：阿里云，百度云，搬瓦工)
2、使用免费空间会助长惰性(亚马逊:12个月，openshift)
3、经常上github.com
4、Hexo本身的主题资源不少，可以自定义

开始搭建 Hexo 博客

上正菜之前，先要说明，搭建Hexo博客是个简单的懒人博客系统。
所以，在文章最后会讲一下普通博客的创建方式。

环境要求

命令行应用 Git
命令行应用 node

环境搭建

Windows

安装Git

到http://msysgit.github.com/ 下载 Git 是个图形化的安装包

现在页面已经自动导向到 https://github.com/git-for-windows/git/releases/tag/v2.13.2.windows.1 这里了。

网页拉倒最下边会看到Git-x.xx.xx-32(64)-bit.exe 去下相应的版本就行了。安装经验戳我

安装node.js

这个就简单了 安装经验 这里全说了

Mac

推荐到这里先安装个brew，

然后通过brew install xxx安装git和node (注意小写)

安装Git

毕竟会有新手，到http://git-scm.com/downloads这里下载，

windows之所以没说这个链接，主要是怕大家下错了。

安装Node

去官网上下载，选择右边最新版本的下载安装就好了。无脑下一步，jd。

准备

github账号创建（简单注册一个账号就可以了）

新建github的仓库

建立与你用户名对应的仓库，仓库名必须为【your_user_name.github.io】

比如我的用户名LSsoftware，我创建的就是LSsoftware.github.io

大家可以点击链接过去看看。

配置SSH-Key

参考github 配置SSH，链接断了自己百度：github配置SSH。

开始搭建Hexo博客

具体以官方为准

安装

打开命令行，执行 

$ npm install -g hexo-cli

部署

在电脑上找个地方，创建一个文件夹，叫[xxx] 你自己起名字。

使用命令行进入这个文件夹，然后输入命令

hexo init

Hexo随后会自动在目标文件夹建立网站所需要的所有文件。

现在我们已经搭建起本地的hexo博客了，执行以下命令，

然后到浏览器输入localhost:4000看看。

$ hexo g
$ hexo s
命令说明
$ hexo g #生成
$ hexo s #启动本地服务，进行文章预览调试
$ hexo d #将.deploy目录部署到GitHub
$ hexo new "xxxxx" #发表新文章

到这里大家就可以用hexo来写博客了

部署到GitHub上之后可以直接到https://xxxx.github.io访问自己的博客

【https://lssoftware.github.io】

发现了个全的 http://www.cnfeat.com/blog/2014/05/10/how-to-build-a-blog/

下边是怎么创建github普通博客

准备

github账号

创建博客

建立与你用户名对应的仓库，仓库名必须为【yourusername.github.io】

进入仓库页面，点击设置按钮（Settings)

进入设置之后点击下方生成页面按钮（Launch automatic page generator)

在这里生成的就是你的博客首页，随便写点什么

到这里个人网站的主页就出来了。就是index.html。

下面就需要你自己写一些html的页面来填充你的博客了
